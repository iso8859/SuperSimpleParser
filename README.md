# SuperSimpleParser
Very simple command line parser in C#

Simply add CommandLineParser.cs to your project.
See Program.cs for example.

Output is
---

---

f=webmaster@acme.com

b=body.txt
j=report1.pdf, report2.pdf
zip=true
